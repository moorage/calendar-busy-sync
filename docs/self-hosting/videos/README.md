# Booking setup video scripts

Generate local video-script artifacts with:

```bash
./scripts/capture-booking-onboarding-videos
```

The command writes scripts under `artifacts/booking-onboarding-videos/` for:

- GitHub Pages setup
- Cloudflare inbox setup
- Vercel inbox setup
- template customization with an AI coding tool

These are scripts and shot lists, not checked-in recordings. Recordings belong under `artifacts/` and should not be committed.
